# Minesweeper_Game

A simple java game, slightly high coupling, and each mode can be debugged individually.

The front frame is written by Java Swing.

### Basic function

- Single player mode
- Double player mode (local)
- Man-machine battle mode
- Save / Load function
- Hack mode
